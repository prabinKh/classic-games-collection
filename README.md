## Overview
This project showcases two classic games implemented with modern web technologies. Both games are designed to be engaging, visually appealing, and accessible on various devices, from desktops to mobiles. The **Rock Paper Scissors** game allows players to compete against the computer, while the **Tic Tac Toe** game supports two-player gameplay with intuitive controls and winner detection.

## Rock Paper Scissors

### Features
- **Interactive Gameplay**: Choose rock, paper, or scissors to play against the computer.
- **Score Tracking**: Tracks user and computer scores in real-time.
- **Responsive Design**: Adapts seamlessly to desktop, tablet, and mobile devices.
- **Modern Styling**: Gradient backgrounds, smooth animations, and hover effects for an attractive UI.
- **Dynamic Feedback**: Displays win, lose, or draw messages with color-coded backgrounds.

### How to Play
1. Open the game in a web browser.
2. Click on one of the three options (Rock, Paper, or Scissors) to make your move.
3. The computer randomly selects its move, and the result (win, lose, or draw) is displayed.
4. Scores update automatically, and a message indicates the outcome of each round.

## Tic Tac Toe

### Features
- **Two-Player Gameplay**: Alternate between X and O in a turn-based game.
- **Winner Detection**: Highlights the winning combination and disables further moves after a win.
- **Reset Functionality**: Includes New Game and Reset buttons to start over.
- **Responsive Design**: Optimized for phones, tablets, and desktops using flexible units (vmin) and media queries.
- **Modern UI**: Smooth animations, gradient backgrounds, and a clean interface.
- **Smart UI Updates**: Updates the board dynamically with each move.

### How to Play
1. Open the game in a web browser.
2. Player X starts first, clicking any box to place their mark.
3. Players alternate turns between X and O.
4. The first player to align three marks (horizontally, vertically, or diagonally) wins.
5. Use the New Game or Reset button to restart the game.

## Installation
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/your-username/classic-games-collection.git
   ```
2. **Navigate to the Project Directory**:
   ```bash
   cd classic-games-collection
   ```
3. **Open the Games**:
   - For **Rock Paper Scissors**, open `rock-paper-scissors/index.html` in a web browser.
   - For **Tic Tac Toe**, open `tic-tac-toe/index.html` in a web browser.
   - Alternatively, use a local server (e.g., Live Server in VS Code) for a better development experience.
